mue_hash (python) v0.0.1
===========================

Python module for MonetaryUnit's X11 hashing.

Install
-------
Python 2.7 is required as well as a gcc.

    $ sudo python2 setup.py install

Test
-------

After installation, test hash.

    $ python2 test.py
